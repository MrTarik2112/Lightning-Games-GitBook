# ⚡ Lightning Games

> Lightweight, high-performance Electron-based game launcher

---

## 🎮 What is Lightning Games?

Lightning Games is a lightweight, high-performance game launcher that lives in the system tray, providing instant access to 59 handcrafted arcade and puzzle games.

### Features

- ⚡ **Instant Launch** - < 1s from hotkey to playable game
- 🎮 **59 Games** - Arcade, Puzzle, Classic, Strategy, Creative
- 🏆 **100+ Achievements** - Achievement system for every game
- 🎨 **11 Themes** - Neon, Retro, Minimal, Forest, Ocean + more
- 🔔 **System Tray** - Open from anywhere with Ctrl+Alt+G
- 🔊 **Synthesized Audio** - 25+ procedural sound effects

---

## 🚀 Quick Start

### Installation

```bash
# Install dependencies
npm install

# Or with Bun (3-5x faster)
bun install
```

### Run the App

```bash
npm start

# Or with Bun
bun start
```

### Opening the App

- **Hotkey:** `Ctrl+Alt+G` (from anywhere)
- **System Tray:** Click the lightning bolt icon
- **Menu:** Right-click tray icon → Open
